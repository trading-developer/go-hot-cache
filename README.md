# chat gpt client

## Описание

```
Легкий клиент для chat GPT для того чтобы  обрабатывать запросы от других программ 
через HTTP API и создавать описания для товаров на основе введенных текстовых подсказок (prompt). 
Программа использует базу данных SQLite для хранения заданий и их результатов.

```

## Api
https://documenter.getpostman.com/view/23579261/2s9YC7SrG6

## Build

GOOS=linux GOARCH=amd64 go build -o ./chatgpt_linux -a
GOOS=darwin GOARCH=amd64 go build -o ./chatgpt_macos -a

    "item_id": 5
